# Nancy Project Manager

This is a Linear AI Agent managed by Kanban AI platform.

## Commands
- `/start-linear-agent` - Main entry point for Linear tasks

## Template Variables
This agent uses the following variables (configured in platform):
- `{{agent_name}}` - Agent display name
- `{{agent_description}}` - Agent description

## Customization
Edit the files in `.claude/commands/` to customize agent behavior.
